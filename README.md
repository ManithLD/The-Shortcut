# The Shortcut

An interactive visual novel about the ethics of using AI in academics.

## How to Run

1. **Install Ren'Py:** Download from [renpy.org](https://www.renpy.org/).
2. **Download Project:** Clone or download this repository.
3. **Launch:** - Open the Ren'Py Launcher.
   - Select **The Shortcut** and click **Launch Project**.

## Theme
A student's journey from taking the "easy way out" with AI to facing the consequences.
